Random choice
===============

This Android application provide you can import data and randomly choose one.

You have two ways to input data.
1. Import from the csv file.
2. Manually input data one by one.

##What the format of csv is accepted
###Example 1
~~~~~~~~
Apple,Banana,Candy
~~~~~~~~
###Example 2
~~~~~~~~
Apple,
Banana,
Candy
~~~~~~~~
###Example 3
~~~~~~~~
Apple,Banana,
Candy
~~~~~~~~
